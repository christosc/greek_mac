Vim keymap that enables one to write traditional (“polytonic”) Greek imitating the corresponding keyboard layout that exists in Mac OS for polytonic Greek. All diacritics are typed before the corresponding letter. In case of two diacritics the order in which they are combined is not important.

Accents

acute ;
grave ]
circumflex [

Smooth breathing

smooth '
with acute /
with grave =
with circumflex -

Rough breathing

rough "
with acute ?
with grave +
with circumflex _

Iota subscript

iota subscript {
with acute ; {
with grave ] {
with smooth ' {
with smooth and acute / {
with smooth and grave = {
with smooth and circumflex - {
with rough " {
with rough and acute ? {
with rough and grave + {
with rough and circumflex _ {

Diaeresis/Dialytika

diaeresis :
with acute :;
with grave :]

Punctuation

Middle dot (“ano teleia”) ; + . or W
Left quote <<
Right quote >>
en dash --
em dash ---

Symbols

Euro sign: E$
