Vim keymap that enables one to write traditional (“polytonic”) Greek.

The keymap is based on the corresponding Mac OS keyboard layout for polytonic Greek.
All diacritics are typed before the corresponding letter.
In case of two diacritics the order is not important.


Accents
-------
 symbol      | key
------------ | -------------
acute        | ;
grave        | ]
circumflex   | [


Smooth breathing
----------------
 symbol      | key
------------ | -------------
smooth       | '
+ acute      | /
+ grave      | =
+ circumflex | -


Rough breathing
---------------
 symbol      | key
------------ | -------------
rough        | "
+ acute      | ?
+ grave      | +
+ circumflex | _


Iota subscript
--------------
 symbol                   | key(s)
------------------------- | -------------
iota subscript            | {
+ acute                   | ; {
+ grave                   | ] {
+ smooth                  | ' {
+ smooth and acute        | / {
+ smooth and grave        | = {
+ smooth and circumflex   | - {
+ rough                   | " {
+ rough and acute         | ? {
+ rough and grave         | + {
+ rough and circumflex    | _ {


Diaeresis/Dialytika
-------------------
 symbol      | key(s)
------------ | -------------
diaeresis    | :
+ acute      | :;
+ grave      | :]


Punctuation
-----------
 symbol                    | key(s)
-------------------------- | -------------
Middle dot (“ano teleia”)  | ; . or W
Left quote                 | <<
Right quote                | >>
en dash                    | --
em dash                    | ---


Other
-------
 symbol          | key(s)
---------------- | -------------
Euro sign        | E$
